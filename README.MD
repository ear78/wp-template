# wp-template

Custom Wordpress template created from scratch using unsplash for photos, affinity designer/photo for the design mockup and custom icon buildin. Fully responsive website utilizing HTML, CSS, PHP, Apache server with SQL was used in the development process.  Javascript will be added for functionality continuously throughout the build.  This was built and designed over the course of a few days while learning how to create a custom Wordpress theme for the Wordpress marketplace.
